# MVPByJude

Addon de World of Warcraft 3.3.5a para gestión de loot, DKP y otras funciones de raid.

## Descripción

Este addon proporciona las siguientes funcionalidades:

- Gestión de loot y BiS
- Comunicación interna entre usuarios
- Sistema DKP y subastas de artículos
- Roles y configuración de UI

## Archivos principales

- `Core.lua`: lógica principal del addon
- `UI.lua`: gestión de interfaces y ventanas
- `Comm.lua`: comunicación entre usuarios
- `LootPicker.lua`: selección y manejo de loot
- `Config.lua`: ajustes del usuario
- `MVPByJude.toc`: lista de archivos del addon

## Instalación

1. Copia la carpeta `MVPByJude` en tu directorio de addons de WoW Classic.
2. Inicia el juego y habilita el addon desde la pantalla de selección de personajes.
